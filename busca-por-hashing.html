Visualização: Busca por Hashing

A Tabela Hash mapeia chaves para índices usando uma Função Matemática. Ao buscar, ela aplica a mesma função e "pula" direto para o índice correto, sem precisar percorrer a lista inteira. Se houver Colisão (dois números no mesmo índice), resolvemos iterando na pequena lista encadeada daquele índice.

📝 Dica FCC: Complexidade de Tempo O(1) no caso médio. O(N) no pior caso (Muitas colisões). Estruturas: Tabela + Lista Encadeada.
